# Restaurant Manager - AWS Amplify Gen 2

A full-stack restaurant management application built with AWS Amplify Gen 2, React, TypeScript, and Bootstrap.

## Features

- ✅ User Authentication (Sign up / Sign in)
- ✅ Create, Read, Update, Delete Restaurants
- ✅ Real-time Updates across multiple clients
- ✅ Search and Filter by city
- ✅ Export to CSV
- ✅ Responsive Design
- ✅ Type-safe with TypeScript

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **UI**: React Bootstrap, Bootstrap Icons
- **Backend**: AWS Amplify Gen 2
- **Database**: Amazon DynamoDB
- **Authentication**: Amazon Cognito
- **API**: GraphQL

## Prerequisites

- Node.js 18+
- AWS Account
- AWS CLI configured

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Hiqmah7/awsamplifyrestuarant
cd awsamplifyrestuarant

2. Install dependencies:
   ```
   npm install
   ```


3. Start the server:
   - For development: `npm run dev`
   - For production: `npx ampx sandbox`
